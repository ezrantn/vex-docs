    Usage #

    The heart of Vex lies in its straightforward syntax:

    vex [command] [pattern]

    Examples # 

    vex replace "foo:bar=input.txt"

    vex load ":=input.txt"

    vex save ":=output"

    vex filter "Hello=world.js"
